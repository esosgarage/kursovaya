Постановка задачи:
1. Провести анализ функций. Построить графики.
2. Написать программу, которая позволяет делать:
    1) Вычисление значения функции в произвольной точке с учетом области определения
    2) Построение таблицы значений на заданном интервале с указанным шагом
    3) Поиск	экстремумов	функции	на	отрезке	методом анализа производной
    4) Поиск аргумента по заданному значению функции с указанной точностью
    5) Вычисление производной численным методом

Введённые функции в программу:
calculate_function(x) - Вычисляет значение кусочно-заданной функции в точке x.
calculate_value() - Интерфейс для вычисления значения функции в одной точке с вводом от пользователя.
build_table() - Генерация таблицы значений функции на заданном интервале.

Для корректной работы программы, в "Страница свойств проекта" выбрать "Свойства". В появившемся меню найти "Препроцессор" и в строке "Определения процессора" ввести _CRT_SECURE_NO_DEPRECATE.
